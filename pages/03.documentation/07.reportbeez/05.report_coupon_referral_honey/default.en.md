The ReferralHoney Statistics Report is part of the [MailBeez ReferralHoney module](/documentation/mailbeez/coupon_referral_honey/).

The report is giving insights how customers are utilizing the referral coupons and is warning for fraud in case referral coupons are used by existing customers.
